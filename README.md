# Webpage-Clone-CSS-only
I have cloned the home page of a graphic design website(for two main reasons)
1. To practice CSS ofcourse and 
2. It looks cool.

Developed a fully responsive homepage clone of a modern website using only HTML and CSS, focusing on pixel-perfect layout design and clean UI structure. Utilized advanced CSS features including Flexbox, Grid, media queries, transitions, positioning, hover effects, and responsive units to ensure smooth adaptability across desktop and mobile devices. Demonstrated strong frontend development skills in layout structuring, responsive web design, visual styling, spacing, alignment, and modern UI implementation while accurately recreating the original website’s appearance and user experience.
